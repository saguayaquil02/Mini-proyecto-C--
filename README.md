# Sistema de control de ventas
Este es un proyecto desarrollado en **C++** utilizando los temas de programacion vistos. El sistema simula la gestión interna de un negocio mediante el control de inventarios, registro de artículos, procesamiento de ventas en tiempo real y cierre de caja, todo manejado a través dearreglos y funciones específicas.
